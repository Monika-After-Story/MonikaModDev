# Monika After Story (MAS)
Monika After Story это мод для бесплатной игры [Doki Doki Literature Club](https://www.ddlc.moe) от [Team Salvato](http://teamsalvato.com/). MAS основывается на 3-ем акте чтобы создать симулятор вашей вечной жизни с Моникой, включающую в себя новые события!

Пожалуйста, проверяйте страницу с [Релизами](https://github.com/OlegSuperBro/MonikaModDev-RU/releases) чтобы получить самый последний стабильный билд.

Если вы хотите создать свой мод как этот, можете посмотреть наш другой проект: [DDLCModTemplate](https://github.com/therationalpi/DDLCModTemplate).

### Installation

1. Скачайте и установите оригинальную DDLC

2. Перейдите на [страницу с релизами](https://github.com/Monika-After-Story/MonikaModDev/releases).

3. Скачайте необходимую версию (с спрайтпаком (Dlx) или без)

4. Перетащите все файлы из скачаного архива в папку с игрой (В папку где находится DDLC.exe), заменив все файлы

5. Запустите игру

*NOTE: Файлы исходников и файлы скачанные напрямую из репозитория используются для разработки и могут работать не совсет так как ожидается.*

Если вам всё ещё нужна помощь с установкой (в том чесле инструкция по установке для non-steam Mac) посетите [FAQ](./FAQ)
